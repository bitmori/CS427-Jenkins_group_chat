All global documentation associated with the T25 project exists within this
directory.

The following is a listing of all the files that exist within this directory
and some background on these files:

- *contract.odt*: The team contract in its source form, which can be modified
  with programs such as 'OpenOffice' and 'LibreOffice'.
- *contract.pdf*: The exported version of the source version of the contract,
  which is used for presentation purposes.
